# MastodonObserver
Mastodonのタイムラインを監視する[棒読みちゃん](http://chi.usamimi.info/Program/Application/BouyomiChan/)プラグイン

## 概要 | Description
MastodonObserverを適用すると、タイムラインを読み上げるようになります。

## 特徴 | Features
* 読み上げを有効にするタイムラインの指定

## 動作環境 | Requirement
* [棒読みちゃん](http://chi.usamimi.info/Program/Application/BouyomiChan/)
* Mastodonトークン
